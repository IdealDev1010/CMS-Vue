# Pruvious Minimal Starter

Look at the [Pruvious documentation](https://pruvious.com/documentation) to learn more.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# yarn
yarn install

# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:2999`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Check out the [deployment documentation](https://pruvious.com/documentation/essentials/deployment) for more information.
