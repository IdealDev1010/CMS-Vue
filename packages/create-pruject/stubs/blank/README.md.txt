# Pruject Starter

Look at the [Pruject documentation](https://pruject.com) to learn more.

## Development Servers

Start the development servers on `http://localhost:2999` (Pruvious) and `http://localhost:3000` (Nuxt):

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

## Deployment

Deploy the application to a remote server:

```bash
npm run deploy
```

## Mirror

Perform content migration between local and remote servers:

```bash
npm run mirror
```

## Backup

Create and restore local backups:

```bash
npm run backup
```

## Generate

Create an [a]ction, [b]lock, [c]ollection, [s]ettings group, or [v]alidator:

```bash
npm run g
```

## Update

Update all Pruvious dependencies:

```bash
npm run update
```

## CLI

Access the command line interface:

```bash
npm run pruject
```
